# 50 Days of Verilog Challenge 🚀

Welcome to my **50 Days of Verilog Challenge**!  
This repository contains a curated set of Verilog programs covering foundational and advanced digital logic concepts.

> 🎯 Goal: Build consistency, deepen hardware design understanding, and create a strong portfolio for core ECE + VLSI + RTL opportunities.

---

## 📚 What’s Inside?

Each folder represents a day or topic:

- ✅ **Basic Gates & Logic**
- ✅ **Half & Full Adders/Subtractors**
- ✅ **Ripple Carry, Carry Look-ahead, Carry Select Adders**
- ✅ **Multipliers**: 2-bit, 3-bit, 4-bit, 4×3-bit
- ✅ **Binary ↔ Gray Code Converters**
- ✅ **Two’s Complement Adder**
- ✅ **Gates using NAND / NOR**
- ✅ **Verilog Testbenches** for each module

All code is written in clean and modular Verilog, tested with corresponding testbenches.

---

## 🛠️ Tools Used

- 💻 **Verilog HDL**
- 🧠 **VS Code**
- 🧪 **ModelSim / Vivado** for simulation
